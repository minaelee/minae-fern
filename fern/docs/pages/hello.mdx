## About Fern Docs

Fern Docs is a documentation platform that uses a docs-as-code approach, including an auto-generated API reference and SDK snippets.

Complete list of technologies I used to spin up this site:
[Fern Docs](https://buildwithfern.com/learn/docs/getting-started/quickstart)
[GitHub](https://github.com)
[git CLI](https://git-scm.com/book/en/v2/Getting-Started-The-Command-Line)
[MDX](https://mdxjs.com/docs/what-is-mdx/)
[YAML](https://yaml.org/)
[JSON](https://www.json.org/json-en.html)
[OpenAPI](https://openapis.org)
