## About Fern Docs

Fern Docs is a documentation platform that uses a docs-as-code approach. It includes an auto-generated API reference and SDK snippets, which can be generated using an API definition written using the [OpenAPI specification](https://openapis.org/) or Fern's own Fern Definition](https://buildwithfern.com/learn/api-definition/fern/overview).

Complete list of technologies I used to spin up this site:
- [Fern Docs](https://buildwithfern.com/learn/docs/getting-started/quickstart)
- [GitHub](https://github.com)
- [git CLI](https://git-scm.com/book/en/v2/Getting-Started-The-Command-Line)
- [MDX](https://mdxjs.com/docs/what-is-mdx/)
- [YAML](https://yaml.org/)
- [JSON](https://www.json.org/json-en.html)
- [OpenAPI](https://openapis.org)
- [VS Code](https://code.visualstudio.com) 
- [Bash](https://www.gnu.org/software/bash)
